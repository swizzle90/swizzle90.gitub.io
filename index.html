Initial Realease of "TV and Movie Tracker v0.8"

(Once a better name is thought of it will change to v1.0)
(Currently there is only support for TV. Movie support will come later.)

This is a collection of plug-ins, scripts, and CSS snippets for use with the markdown based note editor "Obsidian.md". Everything is configured specifically to provide the functions needed so changing settings may break functionality. If you would like something added or changed I will be happy to make changes.

This tracker is not a stand-alone program. It is a collection
of plug-ins and scripts for the note-taking program Obsidian.md.

Plug-in Plug-in Author Plug-in Functionality
Minimal Theme kepano This is the theme that the tracker uses. Currently this is the only theme that will work with the poster view that is used in the tracker. I am working on getting the poster view working for other themes.
Admonition Jeremy Valentine Admonition allows for collapsible, color-coded, block-styled content.
Banners Danny Hernandez Banners adds a banner image to the top of note pages.
Collapse All OfficerHalf Adds a button to collapse or expand all folders while in the file explorer.
Contextual Typography mgmeyers THIS IS REQUIRED FOR SOME FEATURES OF "MINIMAL THEME". Adds a data-tag-name attribute to all top-level divs in preview mode containing the child's tag name, allowing contextual typography styling.
Dataview Michael Brenan aka. blacksmithgu Dataview allows for complex queries using YAML metadata at the top of each note. This is used for the overall show views along with the lists for actors, genres, etc...
Hider kepano Hide UI elements such as tooltips, status, titlebar and more.
Homepage novov Open a specified note on startup. This is configured to show the trackers "Welcome" page on startup. You can also click the House icon in the left bar.
Minimal Theme Settings kepano Change the colors, fonts, and features of Minimal Theme.
Sortable Alexandru Dinu Wiki-like table sorting.

CSS Snippet Name Function
Bigger Preview Allows for a bigger preview window on hover.
Center Image Allows images to be centered.
Explorer Icons 1 Adds folder and note icons while in the file explorer. Uses built in images.
Explorer Icons 2 Adds folder and note icons while in the file explorer. These icons are more minimal than "Explorer Icons 1" and use a font that has to be installed.
Progress Bar This styles the progress bar that is used in the poster and table views of shows.
